# Rust TaskFlow API

Rust TaskFlow API is a task management REST API built with Rust.

## Project Purpose

This project is created to practice intermediate and advanced Rust concepts such as:

- Structs and enums
- Ownership and borrowing
- Traits
- Error handling
- Async programming
- REST API development
- Testing
- Docker
- CI/CD pipeline

## Tech Stack

- Rust
- Axum
- Tokio
- Serde
- SQLite
- Docker
- GitHub Actions
